# L4Re I2C controller driver

The I2C controller driver provides means to separate access to devices on a
shared I2C bus to multiple statically-configured clients.

# Documentation

This package is part of the L4Re Operating System Framework. For documentation
and build instructions see the [L4Re
wiki](https://kernkonzept.com/L4Re/guides/l4re).
